Image

The following REST endpoint mimics a URL scheme similar to Placeholder.com.

  • /api/image/300/ff7f50

                fetch('http://localhost:3000/api/image/300/ff7f50', { method: 'GET' })
                    .then(res => res.blob()).then(blob => console.log(blob));

Random

Generate a list of records from a map of fields.

  • /api/random
Field Types
  • ANIMAL
  • BIRTHDATE
  • CITY
  • COLOR
  • EMAIL
  • FINANCE
  • FULL_NAME
  • GIVEN_NAME
  • GENRE
  • PARAGRAPH
  • PASSWORD
  • PATHNAME
  • PHONE
  • SEX
  • SURNAME
  • USERNAME
  • UUID
  • ZIP

                fetch('http://localhost:3000/api/random', {
                    method: 'POST',
                    headers: {
                        'Content-Type': 'application/json',
                    },
                    body: JSON.stringify({
                        count: 5,
                        fields: {
                            name: 'GIVEN_NAME',
                            genre: 'GENRE',
                            description: 'PARAGRAPH',
                            price: 'FINANCE'
                        }
                    })
                })
                    .then(res => res.json())
                    .then(data => console.log(data));
